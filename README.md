# Open Tab Backup

Chromeで開いている通常ウィンドウのタブ一覧をまとめてバックアップし、後で別ウィンドウとして復元する最小構成の拡張です。

## 現在の仕様

- バックアップは常に最新1件のみ保持
- 保存対象は通常ウィンドウのURLのみ
- 復元時は現在のタブを消さず、新しいウィンドウ群として復元
- 保存先は `chrome.storage.sync`

## インストール方法

1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパー モード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」を押す
4. このフォルダ `/mnt/d/develop/wsl_workspace/open_tab_backup` を選ぶ

## 使い方

1. ツールバーの拡張アイコンから `Open Tab Backup` を開く
2. `今の状態をバックアップ` を押す
3. 復元したいときに `最新バックアップを復元` を押す

## 補足

- `storage.sync` は Chrome Sync が有効な場合、Googleアカウント同期に乗ります
- `storage.sync` には全体容量や1項目あたりの上限があるため、内部ではバックアップデータを分割して保存しています
- `chrome://` など一部の特殊URLはChrome側の制約で期待通り復元できない場合があります
