# Open Tab Backup

Chromeで開いている通常ウィンドウのタブ一覧をまとめてエクスポートコード化し、別PCへ貼り付けて復元する最小構成の拡張です。

## 現在の仕様

- 保存対象は通常ウィンドウのURLのみ
- バックアップ結果はエクスポートコードとして画面表示
- 別PCではそのコードを貼り付けて復元
- 復元時は現在のタブを消さず、新しいウィンドウ群として復元

## インストール方法

1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパー モード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」を押す
4. このフォルダ `/mnt/d/develop/wsl_workspace/open_tab_backup` を選ぶ

## 使い方

1. ツールバーの拡張アイコンから `Open Tab Backup` を開く
2. PC A で `今の状態からコードを生成` を押す
3. `生成コードをコピー` でコードを控える
4. PC B で `インポートコード` 欄へ貼り付ける
5. `貼り付けたコードから復元` を押す

## 補足

- エクスポートコードにはURL一覧が含まれるため、取り扱いには注意してください
- タブ数が極端に多い場合は、コード文字列も長くなります
- `chrome://` など一部の特殊URLはChrome側の制約で期待通り復元できない場合があります
